# Accionari Simulator

Interactive simulator for accionari concepts.

## Structure

- `index.html` — entry point
- `css/style.css` — styles
- `js/main.js` — bootstrap
- `js/knowledge.js` — domain knowledge loader
- `js/ai-chat.js` — AI chat interface
- `js/glossary.js` — glossary panel
- `js/renderers/` — diagram renderers (flux, forte, mcc, transfer, async, mpp, srvm)

## Usage

Open `index.html` in a browser (requires a local HTTP server for ES modules).
